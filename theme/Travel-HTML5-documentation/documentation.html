How to do stuff

How do i customize the intro.html page?
Edit the /assets/js/js-intro.js file, there you will find "INTRO INITIALISATION AND ANIMATIONS". Here all the objects from intro.html are initialized with a position. The intro2.html page is similar with intro.html but with more animations. You will have to edit /assets/js/js-intro2.js file this time and search "TAB CHANGES" to modify the animations. If you dont want to change the animations then simply replace the pictures found on index.html or index2.html under the tag <!--HOME OBJECTS -->
Where do i change the mouseovers from list pages
Edit the /assets/js/js-list.js file, search for "List Hover Animations"
